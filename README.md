# Word Embeddings in Natural Language Processing (NLP)

This repository demonstrates the concept of **Word Embeddings**, a fundamental technique in Natural Language Processing (NLP) used to convert words into dense numerical vectors while preserving their semantic and contextual relationships. The project explores why word embeddings are more effective than traditional one-hot encoding and illustrates how words with similar meanings are represented closer together in vector space.

The repository covers the core concepts of word embeddings, including dense vs. sparse representations, semantic similarity, context-based learning, vector arithmetic, cosine similarity, and popular embedding techniques such as Word2Vec (CBOW and Skip-Gram), GloVe, FastText, and Doc2Vec. It also includes practical implementations and examples to help understand how machines learn meaningful representations of text for downstream NLP tasks.
